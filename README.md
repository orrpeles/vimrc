Tips for vimrc

* do not use or manipulate vimrc under /etc/vim
* rather you should add vimrc file accordingly:
  * ~/.vimrc
  * ~/.vim/vimrc
* ref: https://www.reddit.com/r/vim/comments/c4u8fe/possible_to_change_location_of_vimrc_in_unix/
